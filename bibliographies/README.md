# Bibliographies

Use this folder for literature lists, annotated bibliographies, and public source collections that support AAB research briefs, standards evidence trace, policy briefs, and framework crosswalks.

## Recommended Entry Fields

- Citation or source title.
- Public URL or DOI.
- Source type.
- Relevant learner context or education sector.
- Key contribution.
- Limitations.
- Related AAB registry record or standards topic.
- Review status.

## Notes

Bibliography entries should not be treated as AAB endorsement. They are source-traced inputs for later review, synthesis, and standards development.
