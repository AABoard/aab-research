# Evidence Table Template

| Source ID | Source title | Source type | Public URL or registry ID | Relevant claim | Evidence strength | Limitations | Review status |
| --- | --- | --- | --- | --- | --- | --- | --- |
| SRC-001 |  | research article / case / policy / framework / report / dataset |  |  | high / moderate / emerging / contextual |  | draft / reviewed / needs revision |

## Evidence Strength Guidance

- `high`: directly supports the claim with strong source quality and clear relevance.
- `moderate`: supports the claim but has scope, method, or context limits.
- `emerging`: useful signal but insufficient for strong conclusions.
- `contextual`: background source, not direct evidence for the claim.

## Review Status Guidance

- `draft`: entered but not reviewed.
- `reviewed`: checked for source trace and public suitability.
- `needs revision`: missing source detail, overstates claim, or requires additional context.
