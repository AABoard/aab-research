# Crosswalks

Use this folder for research artifacts that compare AAB concepts with external frameworks, standards, policies, or assessment models.

## Crosswalk Requirements

Each crosswalk should include:

- Purpose and scope.
- Frameworks or sources being compared.
- Mapping criteria.
- Evidence table or source trace.
- Alignment categories.
- Limitations and non-equivalence warnings.
- Review status.

## Alignment Categories

Recommended labels:

- `direct alignment`: concepts substantially match.
- `partial alignment`: concepts overlap but scope or emphasis differs.
- `adjacent`: related but not equivalent.
- `gap`: no clear counterpart.
- `not assessed`: insufficient evidence or out of scope.

## Caution

A crosswalk is an analytical tool, not a statement that two standards or frameworks are interchangeable.
